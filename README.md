# MainCrafts Internship Project

A responsive multi-page website built using HTML, CSS, and JavaScript as part of the MainCrafts Full Stack Web Development Internship.

## Features

* Responsive Navigation Bar
* Hamburger Menu for Mobile Devices
* Dropdown Services Menu
* Home Page with Hero Section
* About Page
* Contact Page
* Contact Form Validation
* Email Validation using JavaScript
* Interactive Feature Cards
* Sticky Navigation Bar
* Mobile Responsive Design

## Pages

* Home (`index.html`)
* About (`about.html`)
* Contact (`contact.html`)

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts (Poppins)
